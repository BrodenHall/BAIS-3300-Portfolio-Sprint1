# Personal Portfolio Website

This repository contains my personal portfolio website developed as part of **Digital Product Management (BAIS 3300)** at the University of Iowa. The site showcases my background, projects, resume, and technical skills, and demonstrates my ability to design, deploy, and analyze a web-based product.

---

## Project Overview

The goal of this project was to design and deploy a professional portfolio website using modern web practices. The site was built as a static web application and deployed through **Azure Static Web Apps** with a custom domain.

In addition to front-end development, the project emphasizes product thinking, accessibility, analytics, and deployment workflows commonly used in industry.

---

## Key Features

- Responsive single-page portfolio layout
- Custom domain deployment via Azure
- Google Analytics integration with campaign tracking
- Cookie consent popup and legal compliance pages
- WCAG 2.1 Level AA accessibility validation
- HTML5 and CSS3 validation
- Downloadable resume and external profile links
